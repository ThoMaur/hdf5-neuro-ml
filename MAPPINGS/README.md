
# Mappings

This directory defines mappings between hdf5-neuro-ml and existing data formats.

## Purpose

- Enable interoperability
- Support data conversion
- Facilitate clinical integration

## Planned Mappings

- DICOM EEG → hdf5-neuro-ml
- EDF / EDF+ → hdf5-neuro-ml
- BrainVision → hdf5-neuro-ml

## Concept

Mappings should preserve:

- Signal integrity
- Timing information
- Metadata consistency

## Future Work

- Conversion scripts
- Validation tools
- Round-trip testing
